# 自己紹介サイト

開発者の名刺的なシンプルな自己紹介サイト。

## 概要

GitHub Pages で公開する自己紹介サイトを作成します。
名刺的に、URL を共有して自己紹介できるようにします。

## 要件

### 必須機能

- [ ] 名前 / ハンドルネームの表示
- [ ] 簡単な自己紹介文
- [ ] SNS リンク（GitHub, Twitter など）
- [ ] プロフィール画像

### あると嬉しい機能

- [ ] スキル・使える技術の表示
- [ ] アニメーション効果

## 使用技術

- HTML/CSS
- React
- GitHub Pages（ホスティング）

## デザイン

### コンセプト

Apple っぽいミニマルなデザインで、グラスモーフィズムを取り入れたい

- 背景: #ffffff
